Welcome to the `create-ts-ddd-esm` documentation.

This template is designed to help you build scalable TypeScript applications using Domain-Driven Design (DDD), modern ESM support, and a monorepo architecture. It supports fast development workflows, proper separation of concerns, and encourages clean architecture practices from the start.

Use this space to get familiar with the structure, motivation, and goals of this starter.
